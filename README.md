
# Simple Portfolio Site

A clean, responsive single‑page portfolio for GitHub Pages.

## Structure
```
.
├── index.html
└── assets
    ├── css
    │   └── style.css
    ├── js
    │   └── script.js
    └── img
```

## Local preview
Just open `index.html` in a browser (double‑click it).

## Deploy on GitHub Pages
1. Create a public repository on GitHub.
2. Upload all files (keep `index.html` at the repo root).
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set **Source** to *Deploy from a branch*.
5. Choose the `main` branch and **/(root)** folder, then **Save**.
6. Wait 1–2 minutes for deployment. Your site will appear at:
   `https://YOUR_USERNAME.github.io/REPO_NAME`

## Customize
- Replace the placeholder text in `index.html` (name, links, email).
- Adjust colors in `assets/css/style.css` (see `:root` CSS variables).
- Add your own images to `assets/img/` and reference them in HTML.
